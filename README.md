# live-commerce-analysis
直播带货消费者行为分析项目：基于K-modes聚类的主播特征挖掘和基于灰色关联度的营销策略分析，是《主播"塌房"下消费偏好与带货提质策略》研究的重要组成部分。

## 项目背景
随着抖音等平台直播电商的快速发展，主播"塌房"事件频发，消费者信任度下降。本项目旨在通过数据分析方法：
1. 识别不同类型消费者的主播偏好特征(K-modes聚类)
2. 量化各类营销策略对购买行为的影响程度(灰色关联分析)

## 数据说明
数据来源：自主设计的问卷调查(有效样本540份)
数据字段：
  - 人口统计学特征：性别、年龄、学历等
  - 消费行为特征：购买频率、偏好主播类型等
  - 营销策略评价：9类策略的评分(1-5分)
数据处理：
  - 分类变量数值化编码(如性别：男=1，女=2)
  - 多选题采用独热编码
  - 数据标准化处理(灰色关联分析要求)
