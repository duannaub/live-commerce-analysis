# live-commerce-analysis
直播带货消费者行为分析项目：
基于K-modes聚类的主播特征挖掘和基于灰色关联度的营销策略分析，是《主播"塌房"下消费偏好与带货提质策略》研究的重要组成部分。

### 1. K-modes聚类分析
​**目标**：识别不同消费者群体的主播偏好
​**方法**：
  - 清洗540份问卷数据，对分类变量（性别/学历等）数值化编码
  - 使用肘部法则确定最佳聚类数K=3
  - 通过`kmodes`库实现分类数据聚类
- ​**结果**：
  - 划分出3类典型群体：娱乐型/知识型/性价比型
  - 输出聚类中心特征表及可视化图

### 2. 灰色关联分析
 ​**目标**：量化9种营销策略对购买行为的影响
​**方法**：
  - 构建影响因素序列（促销活动/商品质量等）
  - 计算各因素与购买行为的灰色关联度
 ​**结果**：
  - 关键因素排序：促销吸引力(0.938) > 商品性价比(0.896) > 商品质量(0.862)
  - 提出策略优化建议（如强化限时促销）
